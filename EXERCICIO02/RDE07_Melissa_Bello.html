Branch

Branch são as modificações em um projeto software ou documento, seu objetivo é o

desenvolvimento paralelo destes projetos.

Geralmente usado para:

  • Novas funcionalidades;
  • Coreeção de problemas;
  • Alteração de código.

São dois tipos de Branch, a BRANCH MASTER, que podemos dizer que é o projeto original e estavel.

BRANCH SECUNDÁRIA, que são as versão usadas para realizar as alterações e melhorias.

Várias pessoas podem fazer alterações em um site por meio de Branchs secundárias, porém só depois de estar tudo ok

que será realizado o Merge, para que as alterações possom entrar no ar.

Hospedagem

Para que um site esteja acessível na internet, é preciso que ele esteja hospedado em algum lugar que ofereça esse serviço.

A maiorio das hospedagem é paga, geralmente com um valor mensal, tipo um aluguel para que seu site fique no ar.

Com a GitHub é possível ter essa hospedagem gratuítamente.
