Hiperlink Externo

O hiperlink externo serve para você linkar outros endereços web na sua página. É muito útil quando você quer redirecionar quem está lendo para outra página web.

Abaixo você pode ver como é a estrutora do comando.

Hiperlink Local

O hiperlink local é uma grande ferramenta quando o seu objetivo é redirecionar o leitor para outra página ainda dentro do site.

Um bom exemplo de hiperlink local é o site Wikipédia, todas as palavras linkadas levam a novas páginas já existentes no site
